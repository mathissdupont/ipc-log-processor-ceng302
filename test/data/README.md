# test data

Bu klasor otomatik testlerde kullanilan log fixture dosyalarini tutar.

## Dosyalar

- `web_server_logs.txt`
- `db_server_logs.txt`

## Kullanim

Bu dosyalar `scripts/test_script.sh` tarafindan veri kaynagi olarak kullanilir.
Script varsayimi geregi test calisma aninda dosyalarin calisma dizininde gorunmesi gerekir.

## Not

Orijinal test verilerini degistirmeden saklayin. Farkli senaryolar icin kopya dosya olusturun.
