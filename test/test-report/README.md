# test-report

Bu klasor `feature/test-report` kapsamindaki test raporlari icindir.

## Rapor Icerigi

Her raporda asagidaki bilgiler bulunmalidir:

- Test tarihi ve ortami
- Calistirilan komutlar
- Derleme sonucu
- Pipe test sonucu
- SHM test sonucu
- Gozlenen hatalar ve cozum notlari

## Onerilen Dosya Adlandirmasi

- `report-YYYY-MM-DD.md`
- Ornek: `report-2026-04-23.md`

## Onerilen Rapor Basliklari

1. Ozet
2. Ortam Bilgisi
3. Derleme Sonucu
4. Pipe Test Sonuclari
5. SHM Test Sonuclari
6. Hata Analizi
7. Sonuc

## Not

Terminal ciktilarini eklerken kisa ve anlamli kisimlari alin; gereksiz tekrarlar raporu sisirmesin.
